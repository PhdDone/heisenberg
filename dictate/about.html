Introduction

You can use this web application to enter English text via speech. The web application has been developed within the Estonian national programme for language technology (2011-2017) in the Institute of Cybernetics at Tallinn University of Technology. The developers are Tanel Alum√§e (server, speech recognition) and Kaarel Kaljurand (Javascript module that communicates with the server).

Technology

The application is based on open source software:

  • Kaldi toolkit - speech recognition engine
  • kaldi-gstreamer-server - server software that passes speech signal to Kaldi and recognition results back to the client
  • dictate.js - Javascript module that records speech and communicates with the server
